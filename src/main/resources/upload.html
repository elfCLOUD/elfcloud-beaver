Uploading data

elfcloud.fi Beaver offers you multiple ways to store data to elfcloud.fi service. Files and folders can be sent by using the file selection dialog. You can also drag files directly from the file browser of your operating system and drop them to elfcloud.fi Beaver to store them to elfcloud.fi.
  • Uploading data using the file selector
  • Uploading data by using drag and drop
  • Upload progress

Uploading data using the file selector

Files and folders can be stored to elfcloud.fi by selecting them using the file selector tool from the toolbar. You can browse your local file system and select the files and folders you want to upload. The files and folders will be stored with the same names as they have on the file system.

Files that you send will be encrypted using the currently active encryption key. You can see the currently active encryption key from the bottom of the elfcloud.fi Beaver interface (see Fig. 1).

Figure 1. Indication of the active encryption key.

Uploading data by using drag and drop

Files and folders can be stored to elfcloud.fi by dragging them from the local file system browser. You can browse your files using the file browser of your operating system and select the files and folders you want to send. When you drag the selection to elfcloud.fi Beaver, your data will be stored to elfcloud.fi service. The files and folders will be stored with the same names as they have on the file system.

Files that you send will be encrypted using the currently active encryption key. You can see the currently active encryption key from the bottom of the elfcloud.fi Beaver interface (see Fig. 1).

Upload progress

Upload progress can be monitored from the dialog that appears when you start uploading files. Current upload task can be cancelled by pressing the Cancel-button.

If a file with the same name as the uploaded file already exists, you will be prompted to select the action you want to perform.

If a folder has the same name as the uploaded file, the file will be skipped.
