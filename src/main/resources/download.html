Downloading data

Downloading your data from elfcloud.fi service is fast and easy. Files downloaded are automatically decrypted with the correct encryption key (if the key is listed in your key manager).
  • Downloading selected files
  • Downloading all files
  • Download progress

Downloading selected files

elfcloud.fi Beaver allows you to download only the files you want. You can select files and folders in the elfcloud.fi Beaver folder browser and select the Download -tool from the toolbar. This will bring up a destination selection dialog. Selected files and folders will be saved to the selected destination.

Downloading all files

elfcloud.fi Beaver allows you to quickly download all content in a directory of your choice. You can do this by selecting Download -tool from the toolbar when you don't have anything selected. This will bring up a destination selection dialog. All content from the current directory will be saved to the selected destination.

Download progress

Download progress can be monitored from the dialog that appears when you start downloading files. Current download task can be cancelled by pressing the Cancel-button.
