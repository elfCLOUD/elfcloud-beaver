Managing folders

elfcloud.fi Beaver allows you to easily manage the structure of your your data stored in elfcloud.fi service.

  • elfcloud.fi data structure
  • Adding top level folder (Vault)
  • Adding sub-folder (Cluster)

elfcloud.fi data structure

All data in elfcloud.fi service will be stored in Vaults, which is the top level container for data. Each Vault has a type that specifies which clients have an access to the given Vault. elfcloud.fi Beaver has an access to two (2) vault types: "com.elfcloud.datastore" and "com.elfcloud.backup".

Adding top level folder (Vault)

Vaults can be added from elfcloud.fi Beaver by using the folder addition tool from the application toolbar when elfcloud.fi Beaver is in the Vault Directory (see Fig. 1).

Figure 1. Vault Directory

Clicking the folder addition tool brings up a dialog (see Fig. 2) in which you only need to provide a name for the Vault and select the type from the dropdown menu. After clicking the Add-button your new folder will be added to the Vault Directory list.

Figure 2. Vault addition dialog

Adding sub-folder (Cluster)

Clusters can be added when elfcloud.fi Beaver is inside a Vault. Clusters can be added in a similar way to adding Vaults. Clicking from the application toolbar brings up a dialog in which you only need to input a name for the Cluster. Clicking the Add-button adds a new folder inside the current folder.

Folders can also be added by dragging them from the file system and dropping inside the elfcloud.fi Beaver (see Fig. 3).
